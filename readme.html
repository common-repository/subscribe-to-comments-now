Subscribe to Comments plugin has "subscribe without commenting" form, but you must edit your template to insert it. This plugin displays this form.

Plugin Homepage on WordPress.org: Subscribe to Comments Now!

Author Homepage: Poradnik Webmastera

Author: Daniel Frużyński

Description

Subscribe to Comments plugin has "subscribe without commenting" form, but you must edit your template to insert it. This plugin displays this form without the need to edit anything. Just download and activate it - that's all :)

Available translations:

  • English
  • Polish (pl_PL) - done by me
  • Russian (ru_RU) - thanks Fat Cow
  • Italian (it_IT) - thanks Gianni Diurno
  • Dutch (nl_NL) - thanks Rene
  • German (de_DE) - thanks Rian Klijn
  • Danish (da_DK) - thanks Torben Bendixen

Installation

  1. Upload subscribe-to-comments-now directory to the /wp-content/plugins/ directory
  2. Activate the plugin through the 'Plugins' menu in WordPress
  3. Enjoy :)
