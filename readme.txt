== Description ==

[Subscribe to Comments](http://wordpress.org/extend/plugins/subscribe-to-comments/) plugin has "subscribe without commenting" form, but you must edit your template to insert it. This plugin displays this form without the need to edit anything. Just download and activate it - that's all :)

Available translations:

* English
* Polish (pl_PL) - done by me
* Russian (ru_RU) - thanks Fat Cow
* Italian (it_IT) - thanks [Gianni Diurno](http://gidibao.net/)
* Dutch (nl_NL) - thanks [Rene](http://wordpresspluginguide.com/)
* German (de_DE) - thanks [Rian Klijn](http://www.creditriskmanager.com/)
* Danish (da_DK) - thanks [Torben Bendixen](http://www.freelancekonsulenten.dk/)

[Changelog](http://wordpress.org/extend/plugins/subscribe-to-comments-now/changelog/)

== Installation ==

1. Upload `subscribe-to-comments-now` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy :)
